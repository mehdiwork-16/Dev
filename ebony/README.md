# EBONY Lounge — Premium Nightclub Website

A full Next.js 14 website for EBONY Lounge. Built with Tailwind CSS and Framer Motion.

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **Tailwind CSS**
- **Framer Motion** (page transitions, scroll reveals, hover effects)
- **TypeScript**

## Prerequisites

- Node.js 18+ — download at https://nodejs.org

## Getting Started

```bash
# Navigate to the ebony folder
cd ebony

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — Hero, Events preview, About preview, Featured drinks, CTA |
| `/events` | Events — Filterable event cards |
| `/reservation` | Reservation — 3-step booking form |
| `/menu` | Menu — Filterable cocktails & spirits |
| `/about` | About — Story, values, gallery, team |
| `/contact` | Contact — Form, hours, social links |
